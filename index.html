Total Plagiarism Checker.

A Plagiarism Scanner developed in Python that supports a website built in HTML5, CSS3 and JavaScript hosted on a Django Server. It's codebase is written in Python and uses machine learning to understand documents' context. The AI is implemented using Gensim - Topic Modelling for Humans and Nltk. It also uses RabinKarp String Search algorithm to compare between two documents. The Plagiarism Scanner can scan documents with the internet and find matching content for plagiarism detection using Webhose.io API. A chatbot has developed been for the same purpose.
